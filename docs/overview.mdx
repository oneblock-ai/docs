## Overview

**1Block.AI** is an open-source, cloud-native LLMOps platform that empowers innovation in LLMs and generative AI applications. It is built on top of cutting-edge technologies such as [Kubernetes](https://kubernetes.io/), [Ray.io](https://ray.io/), [LangChain](https://python.langchain.com/docs/get_started/introduction), etc., and designed to be cloud-agnostic and ML framework agnostic.
